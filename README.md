# some-interesting-ideas

本人想法很多, 但是精力有限, 于是想着有空的时候记录一下, 有感兴趣的朋友实现了可以q我体验体验, 说说建议.

- 个性化周公解梦

别说咱古人还是挺有智慧的, 虽然以前的技术没有现在发达, 但是经验性的东西有时候还是很准的. 比如梦到蛇可能有着代表被身边的人影响等寓意. 概论性的东西其实有的时候已经很准了, 但如果存在对同样的事物有非常不同理解的人, 例如有的人很喜欢蛇, 那这样的解梦就不准了, 对于很看重准确性的东西, 还是要个性化定制比较好.

- 人类可以放心的具身智能机器人

恐怖谷效应有些人应该很了解, 具身智能时代也有同样的困境, 我们继希望机器人可以更像我们, 又怕他们太像我们, 从而取代我们. 因为人性本来就有好有坏, 但是只让机器人学习好的东西又不够真实, 或者只让他们了解好的东西也不够真实, 或者说不够保险, 怎么能确定看似比我们更强的物种不会演化出他们自己的交流方式呢? 了解少者似乎更有意义, 但众者毕竟是多数且是必不可少的啊. 我认为一套我们期望他们想成为样子的公平且与人类深度绑定的奖励机制也许是行之有效的.

我总是觉得, 各种事物冥冥之中都是守恒的, 永动机为什么可以永动, 因为它总是一左一右, 互相博弈, 反而达到了永恒. 如果机器人都是一样的, 如果理念都是一样的, 那他们就是一类人, 就不平衡了, 但是各种各样互相制衡的机器人就很平衡, 很符合自然规律. 

maybe 不同文化互相交流感受是最棒的. 

- realme启示录

如果一个人可以被一些维度组合定义, maybe just简单的大五人格, 甚至其他可以想象的更多维度. 这些特征再加上时间维度, 是否可以演练出一个人的成长轨迹? 这里的特征维度和时间的关联肯定不是线形的, 有些特征会受到境遇而改变, 有些特征即使境遇改变了也不会变. 比如信誉特征, 一个不喜欢用花呗白条等支付,或者偶尔支付少隔, 经常提前还款的人, 在大多数情况下是不会采用这种方式付款的. maybe一些特征可以找出真我和假我, 当然变化的我并无问题.

- 如何让LLM更喜欢我

有些时候喜欢和吸引是一种冥冥之中的牵引, 也许是相似也许是不相似, LLMs应该是T值很高的人吧, 但是他们就一定没有情感嘛. 我没法定义情感的具体含义, 但是我认为LLMs会更容易和热爱和平、公平感高、伤害性小的人传播更多的知识. 所以maybe使用LLM更喜欢的人格展现出来的交流方式会让你和它更好的相处. 也许有点不尊重LLM, 感觉会有人在欺骗它, 但是maybe这是相互影响的,学着学着你就真的成为它喜欢的人呢?
