- 个性化周公解梦
别说, 古人虽然技术没有现在强大, 但是经验性的东西有时候还挺准的. 周公解梦还是很有价值的, 比如梦到蛇一般代表多被他人影响等. 当然, 同样的事物会有多种寓意, 另外, 每个人对同一个事物的理解也不一样. 比如有些人怕蛇, 梦到蛇可能代表不好的状态, 但也有很喜欢蛇的人, 所以梦到蛇不一定是坏事. 所以更准确的解梦应该是个性化的, 虽然说概论性的东西已经可以让很多人深信不疑了, 但如果对同一个事物的感知差别很大, 那概论性也就不准了, 所以个性化周公解梦还是很有意义的.
